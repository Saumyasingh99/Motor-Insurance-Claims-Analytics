# 🚗 Motor Insurance Claims Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard for analyzing motor insurance data. The dashboard provides insights into insurance claims, premium collections, renewals, cancellations, profitability, vehicle characteristics, and customer payment trends.

The objective is to help insurance companies monitor business performance, identify claim patterns, and support data-driven decision-making.
